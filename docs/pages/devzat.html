devzat: an ssh chat room

A project made by Ishan Goel
To get into devzat all you have to do is open command prompt or a terminal if on linux.
The quick way of connecting is entering the command ssh username@devzat.hackclub.com -p 443
be sure to replace username with what you want to be called and congrats.
